# js-100days

JavaScriptの練習用。
小さいアプリを毎日作る！
2026/02/05開始。

##ログ

-Day00(2/6):
GitHubのリポジトリを作って、pushできるようになった！

-Day01(2/5):
ToDoアプリと、背景の色を変えるアプリを作った。

-Day02(2/6):
やる気スイッチアプリ。
ボタンを押すと、やる気ON/OFFを切り替えられる。
やる気ONにすると、カウントアップタイマーが始まる。
やる気をOFFにすると、タイマーがリセットされ、前回の作業時間が下に表示される。

-Day03(2/7):
ビンゴカードアプリ。
アルファベットが並んでいる。マスをクリックしてビンゴを作ると、ビンゴ列のマスの色が変わる。

-Day04(2/8):
マルバツクイズアプリ。
キッズニフティの「インターネットクエスト」の模倣を試みた。
UIはまだだけど、スタート画面→クイズ→結果発表の流れは作ることができた。

-Day05(2/9):
アルファベットカード。
Day03のビンゴカードを作り直して、アルファベットが26文字ランダムに表示されて、クリックすると消えるものを作った。
大文字と小文字を切り替えられるアイデアを取り込んだけど、切り替わる際にシャッフルされないほうがいいかと思い、切り替えボタンとシャッフルボタンを分けることにした。でも完全には上手く動いていない（急に整列しちゃったりする）。

-Day06(2/10):
小学生向けの教材を配布するためのサイトを作った。
.jsファイルを分けること、jsでhtmlの要素をがっつり作ることを勉強した。
ロゴや綺麗なUIにこだわったけど、まだまだやることはある。教材のタグ付けとか、メニューバーとか、カードの大きさ指定とか。教材に番号を振ったほうがいいかな？とも思った。
もう少し整えたら別アカウントで公開したい。
